## Get the code
```
git clone https://github.com/askcarter/infoshare
go version
go get google.golang.org/grpc
```

## Test it out
```
cd infoshare
go run client.go 
go run client.go list
```

## Run the node server
```
npm install grpc
node server.js 
```

## Run the go client
```
go run client.go
go run client.go list
```
